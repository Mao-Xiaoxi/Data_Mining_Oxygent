# 华东师范大学2025数据挖掘期末项目

| 模块大类           | 功能模块                    | 核心职责                                               | 对应任务示例                                                 |
| :----------------- | :-------------------------- | :----------------------------------------------------- | :----------------------------------------------------------- |
| **信息感知与处理** | **多模态识别模块**          | 识别和处理图像、音频、视频中的信息。                   | 1. 识别图片中的人物（张伟丽、妇好）或数字。 2. 解析视频，定位特定时间点的内容或鼠标操作。 3. 识别歌曲、提取歌词或分析音频特征。 |
|                    | **文档解析模块**            | 解析和提取PDF、本地文件系统中的结构化与非结构化信息。  | 1. 从PDF中提取文本规则或图片中的文字。 2. 统计指定目录下的文件数量和类型。 |
| **数据获取与提取** | **网络爬虫与数据提取模块**  | 自动抓取、解析和提取网页、API中的动态与静态信息。      | 1. 抓取京东商品详情、价格、评论。 2. 获取GitHub仓库的Issue、Commit、Release信息。 3. 查询百度百科等知识库内容。 |
| **核心分析与计算** | **逻辑推理与计算模块**      | 执行数学运算、逻辑推理、算法实现和代码生成。           | 1. 解决“毒桶问题”、“金条切割”等逻辑题。 2. 计算投资回报、汇率转换。 3. 生成并执行NumPy/Python代码。 |
|                    | **知识检索与问答模块**      | 基于内部知识库或外部搜索进行信息检索和精准问答。       | 1. 查询历史人物、事件、地理信息（如商朝年数、地级市数量）。 2. 检索企业新闻、技术框架的最新动态。 |
| **系统支撑与协调** | **自然语言理解（NLU）模块** | 理解用户问题的意图、实体和约束，并将其转化为系统任务。 | 1. 解析“按从左到右顺序输出”、“仅输出数值”等复杂指令。 2. 进行实体链接和消歧（如区分不同“苹果”）。 |
|                    | **任务调度与执行引擎**      | 协调各模块，管理复杂任务的工作流、顺序和依赖关系。     | 1. 安排“先爬取网页，再提取数据，最后进行计算”的流程。 2. 处理需要多步骤、多轮交互的复杂问题。 |
